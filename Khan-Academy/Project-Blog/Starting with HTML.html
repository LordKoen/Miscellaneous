Starting with HTML/CSS

A change of style would be good

I have always wanted to learn how to program, I remember trying for the first time when I was 12 years old, watching some Italian guy teaching how to code in C and to practice my skills I tried to make a databank of all my grades which would ask me what grade I wanted to see and from what period. Since then I have tried a few other times with a few other langauges like Java and Python but everytime I would stop after a few days and not come back to it.

I wanted to come back again, I just needed the right incentive. Turns out that incentive was my friend's dad saying he could offer be an intership job if I learned how to create websites. He also gave me a starting place, Khan Academy so that is what I have been doing for the last few days.

So far I have been learning the basics of HTML/CSS, the stuff I have learned so far is:

  • The basic skeleten and tags
  • How to make a list, as you can see :)
  • How to emphasize text using html
  • How to add and deal with Images
  • The different selectors of CSS and how to change their colour

So far I have been really enjoying learning how to make a webpage, it is quite different to what I have tried to learn until now but I think that is a good thing as my last few attempts were not very successful and I hope that I can keep going this time. I am going to learning some more CSS and doing my first website next so I am really excited for that.
