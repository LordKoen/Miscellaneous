Ways of selecting in CSS

Selecting by ID

An item can be selected by ID, to do this go to the item and after it write [id="tagname"]. Then to make changes go into the style tag and write [#tagname] and insert the CSS commands inside.
The problem with this method is that ids have to be different therefore you could not select and whole group of items.
This can also be attributed to tags such as strong and em.

Selecting by Class

Selecting by class is similar to selecting by ID, however they are used when a whole number of items in the same group want to be selected
To use it go for the item and write [class="classname"], and to use it go to the style tag and write [.classname]
The main advantage of this is that the same class can be used for multiple items, sort of grouping them together
